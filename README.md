# 项目简介

这是我的一个练手的开源项目--生活小助手，主要的功能有：

用户注册，登陆；

登陆界面每次登陆都会有一个励志的句子，登陆背景则采用金山词霸的每日一图；

进入app后，首页有三个主要功能：聊天机器人，新闻时事，美女福利。

- 聊天机器人使用网上的免费接口，在使用该接口过程中，发现该接口返回的语句略显粗俗，但是暂时没找到别的接口；

- 新闻时事采用天行数据的新闻接口，点击新闻条目后，内部跳转到新闻详情页面；

- 美女福利采用代码家的图片接口，采用photoview实现大图查看；

首页菜单项提供了手机归属地查询，物流查询和天气查询功能；

在左侧导航菜单可以编辑个人资料，查看软件信息，退出登陆；


# 计划

随着个人技术的进步，这个项目我还会不断完善；目前考虑添加的功能有：股票信息查询，聊天界面个人头像，添加好友。
